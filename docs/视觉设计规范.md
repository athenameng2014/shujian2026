# 书间视觉设计规范

## 1. 设计原则

- **莫兰迪底色 + 高饱和跳色**：大面积保持低饱和的温暖中性色，用少量高饱和色彩作为视觉跳点
- **呼吸感留白**：避免拥挤，每个模块之间保持充足间距
- **色彩即功能**：不同功能模块用不同跳色区分，用户通过颜色即可识别当前所在区域

---

## 2. 色彩体系

### 2.1 莫兰迪底色（大面积使用）

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#faf9f7` | 暖白，非纯白 |
| 卡片/面板 | `#ffffff` | 纯白表面 |
| 主文字 | `#2c2a28` | 暖黑 |
| 次要文字 | `#8a8680` | 暖灰 |
| 边框/分割线 | `#ebe8e3` | 柔和灰 |
| 主强调色 | `#c4956a` | 莫兰迪赤陶（保留但减少使用） |
| 主强调浅色 | `#f5ece3` | 赤陶浅底 |

### 2.2 高饱和跳色（点缀使用）

| 名称 | 色值 | 浅底色值 | 用途 |
|------|------|----------|------|
| Coral 珊瑚红 | `#E8654A` | `#FDE8E3` | 日历页主色、今天按钮、打卡入口、Tab 激活 |
| Ocean 海蓝 | `#3D7C98` | `#E1EFF5` | 搜索按钮、书籍搜索框 focus、专题页标识 |
| Sage 苔绿 | `#5B9E6F` | `#E3F2E8` | 手动添加确认、我的页标识、Tab 激活 |
| Sun 日金 | `#E5A93D` | `#FDF3E0` | 今日阅读标识点、渐变搭档 |
| Berry 浆果紫 | `#9B5DE5` | `#F0E6FD` | 完成按钮、随想输入框 focus 态 |

### 2.3 渐变色（按钮/装饰）

| 场景 | 渐变方向 | 起始色 → 结束色 |
|------|----------|-----------------|
| 浮动打卡按钮 | 135deg | Coral `#E8654A` → Sun `#E5A93D` |
| 搜索按钮 | 135deg | Ocean `#3D7C98` → `#5BA3C4` |
| 确认添加按钮 | 135deg | Sage `#5B9E6F` → `#7BC08A` |
| 完成按钮 | 135deg | Berry `#9B5DE5` → `#C77DFF` |

### 2.4 书籍代表色池（12色，莫兰迪+跳色混搭）

```
#E8654A  #3D7C98  #5B9E6F  #E5A93D
#9B5DE5  #E07A5F  #81B29A  #c4956a
#4ECDC4  #FF6B9D  #F2CC8F  #6C8EBF
```

新建书籍时随机分配，已用过的颜色优先避开。

## 3. 视觉增强点

### 3.1 页面顶部渐变光晕

每个主 Tab 页顶部有一个大面积的径向渐变装饰，营造柔和氛围感，不干扰内容阅读。

| 页面 | 光晕色 | CSS class |
|------|--------|-----------|
| 日历 | Coral 浅色 | `blob-coral` |
| 专题 | Ocean 浅色 | `blob-ocean` |
| 我的 | Sage 浅色 | `blob-sage` |

实现方式：`radial-gradient`，透明度 0.4，高度 48（12rem），`pointer-events-none`。

### 3.2 Tab 栏独立跳色

底部三个 Tab 各有自己的专属颜色，激活态不仅文字变色，还有一个 1px 的圆点指示器。

| Tab | 激活色 | 指示点色 |
|-----|--------|----------|
| 日历 | Coral | Coral |
| 专题 | Ocean | Ocean |
| 我的 | Sage | Sage |

图标在激活态有半透明填充（`fillOpacity: 0.1~0.15`），增强选中感知。

### 3.3 日历格子着色

- 有打卡记录的格子：背景填充第一本书代表色的 12% 透明度版本（`hexToRgba(color, 0.12)`）
- 今日无记录：浅珊瑚红环 + 淡底色（`ring-2 ring-coral/30 bg-coral-light/30`）
- 今日有记录：珊瑚红加粗数字 + 着色背景
- 色点：每个打卡记录显示 5px 圆点，颜色为书籍代表色

### 3.4 卡片彩色左边框

今日阅读 Timeline、打卡记录列表中的每条卡片，左侧有 3px 宽的书籍代表色边框（`border-l-[3px]`），一眼区分不同书籍。

### 3.5 浮动按钮渐变

右下角打卡按钮使用 Coral→Sun 渐变 + 阴影，是页面中饱和度最高的元素，引导用户点击。

### 3.6 搜索/输入框 Focus 态

不同场景的输入框 focus 时显示对应场景的跳色：

| 场景 | Focus 边框色 | Focus 光晕 |
|------|-------------|-----------|
| 书名搜索 | Ocean 50% | Ocean 10% ring |
| 手动输入书名 | Sage 50% | Sage 10% ring |
| 随想文本框 | Berry 50% | Berry 10% ring |

### 3.7 渐变分割线

Bottom Sheet 中的"或"分割线使用渐变（`from-transparent via-border to-transparent`），两端淡出，视觉更轻盈。

## 4. 字体规范

```
--font-sans: "Inter", "PingFang SC", "Noto Sans SC", system-ui, -apple-system, sans-serif;
```

- 页面标题：`text-2xl font-semibold tracking-tight`
- 区域标题：`text-sm font-semibold`
- 正文：`text-sm`
- 辅助文字：`text-xs text-text-secondary`
- 按钮文字：`text-sm font-medium` 或 `font-semibold`

---

## 5. 圆角规范

| 元素 | 圆角 |
|------|------|
| 按钮/输入框 | `rounded-2xl`（16px） |
| 日历格子 | `rounded-xl`（12px） |
| 卡片 | `rounded-2xl` |
| Bottom Sheet | `rounded-t-3xl`（顶部 24px） |
| 浮动按钮 | `rounded-full` |
| 书籍封面缩略图 | `rounded-lg`（8px） |

---

## 6. 阴影规范

仅在有层级关系的元素上使用，保持克制：

| 元素 | 阴影 |
|------|------|
| 日历面板 | `shadow-sm` |
| 月份切换按钮 | `shadow-sm` |
| 今日阅读卡片 | `shadow-sm` |
| 浮动打卡按钮 | `shadow-lg` + 色彩阴影 |
| Tab 栏 | 无阴影，靠 `backdrop-blur-xl` 区分层级 |
