# 书间 V1.0 — 技术方案

> 最后更新：2026-03-31 | 基于 V1.0 已上线实现

## 技术栈

| 层面 | 选型 | 说明 |
|------|------|------|
| 构建工具 | Vite 8 | 极速开发体验 |
| 前端框架 | React 19 + TypeScript (strict) | 类型安全 |
| 样式方案 | Tailwind CSS v4（原子化 + 自定义主题） | 无 UI 组件库，纯手写组件 |
| 路由 | React Router 7 | Tab 导航 + 子页面 |
| 本地存储 | localForage (IndexedDB) | 结构化数据持久化 |
| 状态管理 | Zustand | 轻量，三个 store（book/log/topic） |
| 封面获取 | Google Books API | 免费、无需鉴权 |
| 可视化 | 纯 SVG + CSS 动画 | 星图组件，无外部图表库 |
| 部署 | Vercel + GitHub | 推送 main 即部署 |

## 数据模型

### Book（书籍）
```typescript
interface Book {
  id: string;            // uuid
  title: string;
  author?: string;
  coverUrl?: string;     // Google Books 封面 URL
  color: string;         // 莫兰迪 5 色自动分配
  isFiction?: boolean;
  createdAt: number;
}
```

### Log（打卡记录）
```typescript
interface Log {
  id: string;
  bookId: string;
  date: string;          // "YYYY-MM-DD"
  note?: string;         // 随想（选填）
  createdAt: number;
}
```

### Topic（专题）
```typescript
interface Topic {
  id: string;
  name: string;
  description?: string;
  bookIds: string[];
  createdAt: number;
}
```

### TopicBook（专题-书籍关联）
```typescript
interface TopicBook {
  topicId: string;
  bookId: string;
  status: 'want' | 'reading' | 'done';
  insight?: string;
}
```

### KnowledgeNode（知识概念节点）
```typescript
interface KnowledgeNode {
  id: string;
  name: string;
  description: string;
  categoryIndex: number;
  linkedBookIds: string[];    // 多对多：哪些书支撑此概念
  recommendedBook?: { title: string; author?: string };
}
```

### KnowledgeCategory（知识分类）
```typescript
interface KnowledgeCategory {
  name: string;
  color: string;              // 分类主题色
  nodes: KnowledgeNode[];
}
```

### StarMapData（星图数据）
```typescript
interface StarMapData {
  topicName: string;
  topicId: string;
  categories: KnowledgeCategory[];
}
```

## 存储层设计（localForage）

键名前缀约定：`book:` / `log:` / `topic:` / `tb:`

所有查询基于 `store.iterate()` + key 前缀过滤，O(n) 扫描。
适合 MVP 阶段，数据量 < 1000 条无性能问题。

## 状态管理（Zustand）

| Store | 状态 | 说明 |
|-------|------|------|
| `useBookStore` | `books[]` | 书籍 CRUD，自动分配代表色 |
| `useLogStore` | `logs[]`, `bookIdsWithLogs` | 打卡记录，按月加载，含"有打卡的书 ID 集合" |
| `useTopicStore` | `topics[]`, `topicBooks[]` | 专题 + 专题-书籍多对多关系 |

## 知识星图实现方案

### 可视化
- 纯 SVG，viewBox `0 0 400 400`
- 三层径向布局：中心（专题名）→ 分类星系（半径 55 虚线圈，距中心 130）→ 概念节点
- 分类色：ocean (`#3D7C98`)、coral (`#E8654A`)、berry (`#9B5DE5`)、sage (`#5B9E6F`)
- 已探索节点颜色与所属分类圈颜色一致

### 节点状态
- 未探索：灰色半透明 + 呼吸脉冲动画
- 已探索：分类色实心 + 辉光动画
- 高密度（linkedBookIds > 1）：外加虚线同心圆光环

### 数据绑定（当前 Mock 阶段）
- `src/data/mockStarMap.ts` 定义 3 本 Mock 书 + 17 个概念节点的多对多关系
- 匹配策略：先按书名关键词匹配，匹配不到按顺序自动分配
- 导出函数：`resolveMockBookIds()` / `getBookLinkedConcepts()` / `getNodeLinkedBooks()`
- 后续替换为 AI 自动生成，只需替换此文件的数据源

### 交互
- 内联预览 + 全屏拖拽/缩放模式
- 节点点击 → ConceptSheet 底部抽屉（多书卡片 + 概念定义）
- 书籍列表显示概念 Tag（分类色胶囊标签）
